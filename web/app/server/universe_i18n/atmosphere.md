## Installation
```sh
$ meteor add universe:i18n
```
### Typescript
```sh
$ meteor npm install --save @types/meteor-universe-i18n
```
## Usage
The package is dedicated to work with React but you can use it without it as well.

```
import i18n from 'meteor/universe:i18n';
```
